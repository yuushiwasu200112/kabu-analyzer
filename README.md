# 📊 Kabu Analyzer

株式投資分析ツール。証券コードを入力するだけで、財務データの取得からスコアリングまで全自動で行います。

## 機能
- EDINET APIから有価証券報告書を自動取得
- 収益性・安全性・成長性・割安度の4カテゴリでスコアリング
- 投資スタイル×投資期間に応じた重み付け
- レーダーチャート・ゲージチャート・時系列チャート・ローソク足
- 強制フィルター（債務超過・赤字継続等の警告）

## 対応銘柄
7203（トヨタ）, 6758（ソニー）, 9984（ソフトバンクG）, 8306（三菱UFJ）, 6861（キーエンス）

## セットアップ
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # APIキーを設定
streamlit run app.py
```
